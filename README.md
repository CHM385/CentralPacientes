# Central de Pacientes

## Descripción del proyecto
Este proyecto fue desarrollado en Java utilizando estructuras de datos lineales enlazadas. El objetivo del sistema es administrar la información de pacientes que son atendidos en distintas clínicas de la ciudad.

El sistema permite registrar, consultar, eliminar y mostrar pacientes utilizando una lista simplemente enlazada.

## Estructura del sistema

El sistema está compuesto por las siguientes clases:

- Paciente: almacena los datos del paciente.
- Nodo: representa un nodo de la lista enlazada.
- ListaPacientes: gestiona la lista enlazada de pacientes.
- CentralPacientes: clase principal que ejecuta el programa.

## Funcionalidades

El sistema permite:

- Registrar pacientes
- Buscar pacientes por ID
- Eliminar pacientes
- Mostrar todos los pacientes registrados

## Estructura de datos utilizada

Se implementó una lista simplemente enlazada donde cada nodo contiene:

- Información del paciente
- Referencia al siguiente nodo

Esto permite insertar y eliminar registros de forma eficiente.
